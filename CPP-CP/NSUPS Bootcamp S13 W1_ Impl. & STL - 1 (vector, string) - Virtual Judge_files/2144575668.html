Two friends Morshed and Abdullah were in a competition to see who could build the magical string first. After a lot of attempts, Morshed was the first one to finish constructing the string, thus winning the competition. After the hard work, he felt tired and went to sleep. Abdullah was in a rage and split Morshed's magical string into two strings named Boga\textit{Boga}Boga and Moga\textit{Moga}Moga (which were initially empty) using the following rule:

  • He traversed Morshed's magical string (starting from its first index), picked the character at the current index, and appended it to either Boga\textit{Boga}Boga or Moga\textit{Moga}Moga. If the index was even, then Abdullah appended that character to the Boga\textit{Boga}Boga string; and if it was odd, then he appended that character to the Moga\textit{Moga}Moga string. Note that here indexing starts from 0. Appending a character to a string means adding it to the back of the string.

Once Morshed was awake he found his magical string split into two. Help him restore his original magical string.
